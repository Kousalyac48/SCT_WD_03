# **Tic-Tac-Toe** ⚡

**Neon Grid** is a high-octane, **cyberpunk-inspired** reimagining of the classic Tic-Tac-Toe. Built with a focus on **sleek UI/UX** and sophisticated logic, it pits human players against an **unbeatable machine intelligence** in a glassmorphic digital arena.

---

### **🚀 CORE FEATURES**

* **🧠 UNBEATABLE MINIMAX AI:** Powered by a recursive **Minimax Algorithm**, the AI evaluates every possible move sequence to ensure **perfect play**. It is mathematically impossible to beat—the ultimate test for a player is forcing a draw.
* **📜 LIVE STEP TRACKER:** A specialized sidebar logs every move with precision, displaying **Step Numbers**, **Player Identity**, **Grid Coordinates (Row, Col)**, and **Real-time Timestamps**.
* **💎 GLASSMORPHIC AESTHETIC:** A premium UI featuring **backdrop-filter blurs**, neon glows, and a responsive grid that reacts to every move with **high-fidelity animations**.
* **⚡ REACTIVE FEEDBACK:** Real-time status updates and **automatic victory path highlighting** for instant game-state recognition.
* **📱 MOBILE OPTIMIZED:** A fluid, grid-based layout that adapts seamlessly from **4K monitors** to handheld mobile devices.

---

### **🛠️ TECHNICAL STACK**

* **LANGUAGES:** **HTML5**, **CSS3**, **Vanilla JavaScript (ES6+)**.
* **INTELLIGENCE:** Recursive **Minimax Algorithm** with depth-based scoring.

---

### **🧠 STRATEGIC LOGIC**

The AI treats the $3 \times 3$ grid as a search tree, evaluating the terminal state of every possible move. It assigns scores based on the outcome:
* **WIN (AI):** $+10$ (adjusted by depth)
* **LOSS (AI):** $-10$
* **DRAW:** $0$

By **maximizing** its own score and **minimizing** the player's potential score, the **Neon Grid AI** guarantees it will never lose a match.
